# Module 8: Extraction, Notes & Research

> Turn any text into structured, usable knowledge.

## What You'll Learn
This module teaches you to extract data into tables and JSON, create multi-level summaries, convert notes to flashcards, and compare documents.

## Why This Matters
Students deal with mountains of text - textbooks, papers, articles, notes. AI can structure all of it in seconds. This module teaches you how.

## Lessons

| # | Lesson | What You'll Master |
|---|--------|--------------------|
| M8-L1 | Extract to Table | Pull structured data from messy paragraphs |
| M8-L2 | Extract to JSON | Convert any text into clean JSON format |
| M8-L3 | Summarize at 3 Levels | Get 1-line, 1-paragraph, and detailed summaries |
| M8-L4 | Notes to Flashcards & Quiz | Turn notes into revision-ready flashcards |
| M8-L5 | Compare 2 Documents | Find differences and similarities between texts |

## Key Skill
Extraction prompting. Paste text in, get structured data out. Saves hours of manual work.

## Time to Complete
50-75 minutes (5 lessons x 10-15 min each)

## Student Relevance
- Summarize research papers in seconds
- Create exam flashcards from lecture notes
- Extract structured data for projects
