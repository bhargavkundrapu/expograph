# Module 6: Video Scripts

> Write scripts that hook viewers in 3 seconds.

## What You'll Learn
This module teaches you to create video scripts, storyboards, shot lists, captions, and ready-made templates for 30/45/60-second videos.

## Why This Matters
Video content is everywhere. Whether you're making a project demo, a LinkedIn post, or a YouTube short - a good script makes all the difference. AI can write it for you in seconds.

## Lessons

| # | Lesson | What You'll Master |
|---|--------|--------------------|
| M6-L1 | Hook + Script Template | Write attention-grabbing video scripts |
| M6-L2 | Script to Storyboard | Convert a script into a visual plan |
| M6-L3 | Shot List & B-Roll | Plan every shot + supporting footage |
| M6-L4 | Captions & On-Screen Text | Generate captions and text overlays |
| M6-L5 | 30/45/60-Sec Templates | Ready templates for short-form video |

## Key Skill
The Hook-Body-CTA structure. Every great video follows this pattern, and you'll master it.

## Time to Complete
50-75 minutes (5 lessons x 10-15 min each)

## Student Relevance
- Create project demo videos
- Build content for social media presence
- Make explainer videos for college presentations
