## SEC-06: âœ… Good Prompt

Prompt:
```
Write a 60-second video script about "3 free AI tools every
college student should use."

Structure:
- Hook (first 3 seconds): Start with a bold question
- Body: Cover 3 tools with one line each
- CTA: Ask viewers to comment their favorite tool

Tone: Energetic, casual, like talking to a friend.
Keep it under 150 words.
```

---

## SEC-07: Good Output

Output:
```
HOOK: "You're still doing assignments the hard way?"

BODY:
Tool 1: Use ChatGPT to brainstorm ideas in 10 seconds flat.
Tool 2: Use Gamma to turn your notes into stunning presentations.
Tool 3: Use NotebookLM to study smarter with AI-powered summaries.

These three tools saved me hours every single week.

CTA: "Which one are you trying first? Drop it in the comments."
```

## SEC-08: ðŸš€ Upgrade Prompt

```
Write a 60-second video script about "3 free AI tools every
college student should use."

Structure:
- Hook (first 3 seconds): Start with a pattern interrupt â€”
  a surprising stat or a bold myth-buster
- Emotional bridge: One line that connects to the viewer's pain
  (wasting time on assignments)
- Body: Cover 3 tools, one line each, with a specific use case
- CTA: Ask a question + tell viewers to follow for more

Tone: Energetic, casual. Use short punchy sentences.
Max 150 words. Add [PAUSE] markers where I should pause for effect.
```

**What changed**: Added an emotional hook, a pattern interrupt opening, pause markers for delivery, and a two-part CTA that drives both comments and follows.
