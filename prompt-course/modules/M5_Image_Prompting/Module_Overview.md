# Module 5: Image Prompting

> Create stunning visuals with text prompts.

## What You'll Learn
This module teaches you to write image prompts that produce exactly the visual you imagine. Control subject, composition, style, lighting, and character consistency.

## Why This Matters
Whether it's a project thumbnail, social media post, presentation visual, or portfolio image - knowing how to prompt for images is a modern superpower.

## Lessons

| # | Lesson | What You'll Master |
|---|--------|--------------------|
| M5-L1 | Image Prompt Formula | Master Subject + Style + Details + Mood |
| M5-L2 | Composition Control | Control layout, camera angle, and framing |
| M5-L3 | Style & Lighting Control | Set art style, color palette, and lighting |
| M5-L4 | Character Consistency | Keep the same character across images |
| M5-L5 | Iterate & Fix Bad Images | Debug and improve image prompts step by step |

## Key Skill
The Image Prompt Formula: Subject + Style + Details + Mood. Master this and you can create any visual.

## Time to Complete
50-75 minutes (5 lessons x 10-15 min each)

## Student Relevance
- Create thumbnails for YouTube and social media
- Design visuals for project presentations
- Build a portfolio with AI-generated mockups
